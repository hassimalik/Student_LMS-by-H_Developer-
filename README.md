# 🎓 Student-LMS by H_Developer  

A **production-ready YouTube Playlist Course Tracker App** built with modern frontend technologies.  
This app helps students add YouTube playlists as courses, track their progress visually, and manage completed courses — all with a beautiful, animated UI. 🚀  

---

## ✨ Features  
- 📂 Add **YouTube playlists** as courses  
- ✅ Mark videos as watched  
- 📊 Track progress with visual indicators  
- 🏆 Completed courses section in profile  
- 🎨 Modern UI with **Tailwind CSS**  
- 🎬 Smooth animations powered by **GSAP**  
- 🔗 Easy navigation with **React Router DOM**  
- 🌀 Custom **background patterns** made inside React  

---

## 🛠️ Tech Stack  
- **React** ⚛️  
- **Tailwind CSS** 🎨  
- **GSAP** 🎬  
- **React Router DOM** 🔗  
- **YouTube API** 🎥
